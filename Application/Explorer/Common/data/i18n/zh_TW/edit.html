快速鍵 

一般快速鍵：
      ctrl+s 儲存      
      ctrl+a 全選      ctrl+x 剪下   
      ctrl+c 複製      ctrl+v 粘上
      ctrl+z 復原      ctrl+y 反復原
      ctrl+f 尋找      ctrl+f+f 取代
      alt+0 折疊所有   alt+shift+0 展開所有
      esc [離開搜尋,取消自動提示...]
選擇：
      滑鼠選擇——拖動
      shift+home/end/up/left/down/right  
      shift+pageUp/PageDown 上下翻頁選中
      ctrl+shift+ home/end  目前游標移動至頂端/底端
      alt+滑鼠拖曳          塊選擇
 游標：
      home/end/up/left/down/right
      ctrl+home/end 游標移動至文件頂端/底端
      ctrl+p 跳轉至匹配的標籤
      pageUp/PageDown 游標上下翻頁
      alt+left/right 游標移動至行首位
      shift+left/right  游標移動至行首&尾
      ctrl+l 跳轉至指定行
      ctrl+alt+up/down  上(下)新增游標
